Kapitel: Machine Learning

Machine Learning ist ein Teilbereich der Künstlichen Intelligenz, bei dem Algorithmen aus Daten lernen, um eigenständig Muster zu erkennen und Vorhersagen zu treffen.

Grundidee:

  • Algorithmen entwickeln selbstständig Lösungsstrategien
  • Modelle werden mit Daten trainiert statt explizit programmiert
  • Qualität hängt stark von Datenmenge und Datenqualität ab

Lernarten:

  • Supervised Learning:
    • Training mit Eingabe- und Ausgabedaten
    • Ziel: Mapping von Input → Output
    • Typische Probleme: Klassifikation & Regression
  • Unsupervised Learning:
    • Keine vorgegebenen Zielwerte
    • Erkennung von Mustern und Strukturen
    • Beispiele: Clustering, Assoziationen
  • Reinforcement Learning:
    • Lernen durch Belohnung und Bestrafung
    • Agent interagiert mit einer Umgebung
    • Ziel: Maximierung der Belohnung

Typische Algorithmen:

  • Lineare & logistische Regression
  • Decision Trees & Random Forest
  • Support Vector Machines (SVM)
  • k-Nearest Neighbors (kNN)
  • Neuronale Netze / Deep Learning

Kapitel: Regression

Regression ist ein Verfahren im Machine Learning, mit dem kontinuierliche Zielgrößen auf Basis gegebener Daten vorhergesagt werden.

Grundprinzip:

  • Vorhersage einer numerischen Zielgröße (z.B. Preis, Umsatz)
  • Zusammenhang zwischen Variablen wird modelliert

Wichtige Arten:

  • Lineare Regression → Zusammenhang als Gerade
  • Multiple Regression → mehrere Einflussfaktoren
  • Logistische Regression → für Klassifikation

Lineare Regression:

  • Form: y = bx + a
  • a = Achsenabschnitt
  • b = Steigung
  • Ziel: Minimierung der Fehlerquadrate (Least Squares)

Wichtige Kennzahlen:

  • Korrelationskoeffizient (R): Stärke des Zusammenhangs (-1 bis 1)
  • Bestimmtheitsmaß (R²): Erklärte Varianz (0 bis 1)

Beispiel:

  • Vorhersage des Autopreises basierend auf Alter
  • Mit steigender Nutzungsdauer sinkt der Preis

Kapitel: k-Nearest Neighbors (kNN)

kNN ist ein einfaches, aber effektives Klassifikationsverfahren, das neue Datenpunkte anhand ihrer nächsten Nachbarn im Datensatz einordnet.

Grundidee:

  • Ein neuer Punkt wird anhand ähnlicher Daten klassifiziert
  • Ähnlichkeit wird über Distanz (z.B. euklidisch) gemessen

Funktionsweise:

  • Wähle k (Anzahl der Nachbarn)
  • Berechne Distanz zu allen Datenpunkten
  • Wähle die k nächsten Nachbarn
  • Bestimme die häufigste Klasse

Eigenschaften:

  • Einfach zu verstehen und umzusetzen
  • Kein explizites Training notwendig
  • Rechenaufwendig bei großen Datensätzen

Einsatzgebiete:

  • Klassifikation (z.B. Spam-Erkennung)
  • Empfehlungssysteme
  • Mustererkennung
